# twallery

Tweet a picture with a given hashtag -  find it in your private Twallery

### Dataflow:

tweet -> listener -> redis -> website

### Install
`$ mkvirtualenv twallery`

`$ pip install nodeenv`

`$ npm install socket.io`

(missing python bit for now)

### Run

##### Node
